1. Fish

Fish是“the friendly interactive shell”的简称，fish最大的特点就是功能强大，智能并且用户友好。Fish支持语法高亮，自动建议，标签完成等，而且配置十分简单。

1.1 安装

Ubuntu和Debian的安装方法：

1

sudo apt-get install fish -y

Mac下安装方式：

1

brew install fish

1.2 配置

修改fish为默认终端

1
2

chsh -s /usr/bin/fish
fish

fish的配置文件为~/.config/fish/config.fish，fish每次启动会自动加载改文件。你也可以使用

1

fish_config

使用fish提供的web配置界面来配置fish。

1.3 使用

fish支持语法高亮，非法的命令显示默认的红色，命令有效后变成另外颜色。有效的路径用下划线标识

1
2
3
4
5
6

# 非法的命令行
mkd
# 有效的命令
mkdir
# 有效路径下有下划线
cd <u>~/Downloads/test/</u>

fish支持自动补全，输入命令时，在光标右侧显示建议(灰色)，使用ctrl+f或者→(键盘右键)接受建议，alt+→接受部分建议

1

sudo apt <u>i</u><font color="gray">nstall fish -y</font>

1.4 Oh-my-fish

oh-my-fish是Fish的主题和插件管理工具(类似oh-my-zsh)，安装非常简单

1
2

# Installation
curl -L https://get.oh-my.fish | fish

安装完成后，使用omf命令安装theme或plugin

1
2
3
4
5

omf --help # 获取 help 信息
omf theme  # 列出可用的主题
omf list # 列出已安装的theme和plugin
omf install kawasaki # 安装 kawasaki theme
omf install https://github.com/jhillyerd/plugin-git # 安装git plugin

2. tmux

tmux是一个非常优秀的终端复用软件，和GNU Screen类似。它可以在一个终端的多个程序之间自由的切换，保证终端关闭或者非正常掉线，任务也会继续运行，并且还能回到原来的工作环境。简单的说，tmux主要提供2大功能：

  1. 窗口管理和split窗口。tmux启动时创建一个新的session，session初始包含一个window，可以在一个session中创建多个window，也可以将window split为多个pane。
  2. session管理。tmux可以从screen detach分离，并在后台运行，也可以attach到screen。

2.1 安装

Ubuntu和Debian的安装方式：

1

sudo apt install tmux -y

Mac下的安装方式：

1

brew install tmux

2.2 配置

tmux的所有的配置参考我的GitHubtmux-vim-conf，因为linux(ubuntu和debian)与mac使用的tmux版本差异较大，有些配置项修改了，所以tmux.conf.linux和tmux.conf.mac分别用于linux和mac的tmux配置

1
2

# 将对应的配置copy到~/.tmux.conf即可
cp tmux.conf.mac ~/.tmux.conf

# Status bar settings

3. vim

3.1 安装

Ubuntu和Debian安装方式：

1

sudo apt install vim -y

Mac安装方式：

1

brew install vim

3.2 配置及使用

我的vim配置基于 exvim 修改，增加了 a.vim, ctrlp.vim, TabBar等插件，将vim的配置及插件拷贝到~目录即可直接使用，下面主要介绍下插件的使用:

1
2
3
4

# 配置，将 .vim* 拷贝到 ~ 用户HOME目录即可
git clone https://github.com/lzhujian/tmux-vim-conf.git
cd tmux-vim-conf
cp -r .vim* ~

a.vim 用于c/c++头文件和.h文件自动跳转，在vim里使用 :A即可在.h/.cpp文件之间来回切换

NERDTree 侧边的导航栏，快捷键 ctrl-n

TabBar当打开多个文件时，顶部栏展示打开的各个文件，使用 ctrl-h/l左右切换打开的文件，ctrl-x关闭当前tab

ctrlp.vim 快捷键ctrl-p，在vim底部打开buffer，ctrlp匹配输入的文件名，ctrl-j/k上下移动，快速打开文件

exvim 使用vim打开xxx.exvim会在当前目录新建exvim project，exvim包含一系列plugin和快捷键，详细可以参考exvim官方文档
